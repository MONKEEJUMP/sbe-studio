Action ID: `SBE4829-20260704`
Purpose: push verified SBE Studio changes through GitHub and Vercel.

## When To Run

Run only after local verification is green and PAULIEWOOD wants the update live.

## Preflight

```powershell
git status --short
pnpm check:public
pnpm lint
pnpm typecheck
pnpm build
```

## Git Rules

- Stage exact paths only.
- Do not use `git add .`.
- Commit with a concise message.
- Push the active branch only after verification passes.

## Vercel Rule

After push, wait for Vercel deployment to finish. Verify `https://sbe.studio/` in the browser before saying the live site is updated.

## Stop Conditions

- Dirty unrelated files are present.
- Verification fails.
- Vercel deployment fails or points to the wrong project/domain.
- Live site still shows stale content after deployment propagation.

## Evidence To Report

- Commit hash.
- Branch pushed.
- Vercel deployment status.
- Live URL checked.
