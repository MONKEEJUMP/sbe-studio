# SBE Verification Action

Action ID: `SBE4828-20260704`
Purpose: prove SBE Studio changes before claiming they are done.

## When To Run

Run after code, copy, layout, data, or deployment-related changes.

## Commands

```powershell
pnpm check:public
pnpm lint
pnpm typecheck
pnpm build
```

## Browser Checks

For visual work, check:

- homepage desktop
- homepage mobile
- work page if portfolio cards changed
- receipts page if stat cards changed
- contact page if forms or links changed

## Expected Result

- Lint passes.
- TypeScript passes.
- Build passes.
- Public-safety checks pass.
- No obvious visual overlap, broken images, invisible controls, or stale private/proprietary references.

## Stop Conditions

- Build fails.
- A private/proprietary entry appears on a public page.
- A stat/cue-card number does not match Receipts.
- Live deployment cannot be verified when live push was requested.

## Evidence To Report

- Commands run and result.
- Browser routes checked.
- Any residual risks.
