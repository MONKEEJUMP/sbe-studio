# SBE Setup Action

Action ID: `SBE4827-20260704`
Purpose: prepare the SBE Studio repo for local Codex work.

## When To Run

Run at the start of a new SBE thread, after dependency changes, or before moving work into a worktree.

## Commands

```powershell
pnpm install
pnpm dev
```

## Expected Result

- Dependencies install without lockfile drift unless a dependency change is intentional.
- Local dev server starts.
- Codex can browser-check the local site.

## Stop Conditions

- Missing Node/pnpm.
- Install fails.
- Environment variables are required and unavailable.
- The command tries to expose secrets or modify unrelated files.

## Evidence To Report

- Install result.
- Dev server URL.
- Any warnings or blockers.
